# 简介

`magic-scroll`是一个基于[React.js](https://reactjs.org/)的一个自定义滚动条组件.它是用来美化和加强原生滚动条的。

## 灵感来源

灵感来源于我的另一个开源项目基于 vue 的滚动条组件 - [Vuescroll](https://github.com/YvesCoding/vuescroll).

## 特点

- 平滑滚动
- 高度自定义，可以自定义 scrollbar(滚动条),rail（滚动轨道）,container（滚动容器）等等。
- 使用起来简单，只需要用`Scrollbar`包裹需要滚动的内容即可，无需强制设置任何属性。
- 支持检测内容尺寸发生变化。
