## 快速开始

### 使用 npm 或者 yarn 安装

```bash
npm i magic-scroll -S
```

### 在项目中导入并包裹内容

```css
.parent {
  width: 100px;
  height: 100px;
}

.child {
  width: 200px;
  height: 200px;
}
```

```js
import Scrollbar from 'magic-scroll';
import React from 'react';

class BasicDemo extends React.Component {
  render() {
    return (
      <div className="parent">
        <Scrollbar keepBarShow>
          <div className="child" />
        </Scrollbar>
      </div>
    );
  }
}
```
